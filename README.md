# library
Library is an application designed to store book information that the user inputted. The overall structure of the website was set up using HTML and styled with CSS. JavaScript was used to set up user interactivity.

<h4>Features</h4>
  <ul>
    <li>Fixed button to add new book to the library
    <li>User input for book information
    <li>Ability to toggle "read" status for each book
    <li>Ability to remove book from the library
  </ul>
  
<h4>Challenges</h4>
<ul>
    <li>Deciding how to store and display each book object
  </ul>
